What’s inside

Two PDFs: the Prompt Pack (your “do-the-work” prompts) + the Verification Checklist (your “trust-but-verify” sheet).

📌 Prompt Pack topics

  • Financial statement analysis (ratios, working capital, tie-outs)
  • Tax & compliance (tax computation, depreciation)
  • Valuation & modeling (DCF setup, WACC)
  • Audit & controls (substantive procedures, reconciliation review)
  • Accounting entries (complex transactions, year-end adjustments)
  • Budgeting & variance analysis

🧾 Verification Checklist sections

  • Input sanity (company/period/units/objective/framework)
  • Evidence check (sources, no invented citations, dated market data)
  • Math check (recalc 3 critical numbers, rounding consistency)
  • Tie-outs (BS balances, cash flow ties, debits = credits)
  • Assumptions (explicit + consistent + sensitivity)
  • Final rating: Green / Yellow / Red

FAQ

Quick answers for anyone who lands here from LinkedIn.

Is this beginner-friendly?

Yes. It’s built with placeholders and clear instructions—so you can start with foundation-level knowledge and still produce clean work.

Which AI tools does this work with?

Any: ChatGPT, Claude, Gemini, etc. The prompts are written to be tool-agnostic—focus is on inputs + verification.

Can I use this for internship / articleship work?

Use it for learning and drafting, but verify everything. Never share confidential client data in AI tools.
